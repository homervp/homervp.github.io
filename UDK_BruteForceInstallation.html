Forcing UDK to install on Windows xp x64 or similar

UDK will not install on such windows as they are based on Server2003 and not XP, and Epic Games state that WinXP SP3 or higher is required. However this workaround worked for me and many others:


-Start the installer with -progressonly passed as an argument

-Copy the file from http://dl.dropboxusercontent/u/29365870/UnEulaLite.exe (kindly given by GMMan in Steam forums) to the same folder as UnSetup.exe ( /UDK/[UDK version]/Binaries)

-In start menu, go to UDK or Unreal Development Kit -> UDK [version] -> UDK Editor and make a shortcut to it. If you cannot find it, find UDK.exe in the 32 or 64 bit folder of Binaries and give it the parameter "editor" (without quotes)
